Data caps idea

For all this iteration of the now page is meant to be in the month when I’m handing things in for university, I seem to be having quite a few ideas! This one sprang into mind the other week, and it stems from my idea for an implementation of slowing down the internet. Incidentally, that implementation no longer works, as I had to plug in the laptop running it with an ethernet cable as it’s also now running as a local server, which made the “slow” internet hotspot run much too fast!

Anyway, this idea is similar and one I might implement when I get some more time. Instead of allowing the same access to a slower version of the internet, this system uses the idea of data caps that we had maybe 10 years ago. They don’t seem to really be a thing in the UK any more, if you’re on Wi-Fi, you can download/upload an unlimited amount of data all day every day, but I remember when I was a child you couldn’t download or upload more than 20GB a month. It made me (and more my parents!) very conscious of my YouTube habits, and I wonder what the psychological impact of re-introducing such a thing might be. I will try and build an implementation over Christmas and see.
